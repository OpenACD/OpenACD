Description

A cdr dumper that makes a dets back repository of cdrs, one for each day. Also has some helper functions to load the data either to ets, or back into mnesia.

Takes all records in the passed dets file and loads them into mnesia. This is useful if you want to migrate the records from the dets dumper to another dumper. Best if used in a spawn, and when the dets dumper is not running.
