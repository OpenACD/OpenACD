The media manager for handling email.

Behaviours: gen_server.

Description

The media manager for handling email. This starts the email server listener. The listener passes off to the email_media_session callback. When the email_media_session callback is ready for an email to enter queue, it sends a message back here for an email_media to be created and queued.
