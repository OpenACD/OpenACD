Common protobuf conversion functions.

Description

Common protobuf conversion functions. Most take an internal data structure and return a record used by the protobufs.

proplist_to_protobuf/1

proplist_to_protobuf(List::simple_proplist()) -> [#simplekeyvalue{}]

Turns a simple proplist into a list of #simplekeyvalue{}, which is protobuf friendly. The proplist can only have atoms, binaries, or strings as the key. The value can only be atoms, binary, or lists. If the value to anything else, it emits a warning and is skipped.
