Description

Behavior for cdr dumping modules. A cdr dumper is a module that takes the erlang record stored in mnesia and stores it in another place, be it a flat file, database, or the great datastore in the sky. This module handles the communication with mnesia, and merely passes the record to be written to any handlers. If a handler errors or fails, it is uncerimoniously dropped.
