library module for getting, setting, and deleting agent web sessions.

Description

library module for getting, setting, and deleting agent web sessions. It uses cpx_agent_web_connection as a tracking pid for each session. If the session pid dies, the associated session id is removed as well, effectively logging the user out.

Function Details

create/1

create(ReqData::web_request()) -> {ok, cpx_web_session(), web_request()}

Create a new session for the request. If there is an old session, it is destroyed. For a non-destruction ensurance of a session, use get_or_create/1 instead.

destroy/1

destroy(ReqOrId::undefined | string() | web_request()) -> ok | {ok, web_request}

Destroy an session that happens to be associated with the request. If a webmachine request is passed in, appropriate http headers are set to destroy the cookie client side and returned. Otherwise just informs the session pid it should kill itself.
