OpenACD

Version: 2.0.0

OpenACD is an open source automated contact distributer. It is Designed to be media and agent endpoint agnostic. Paired with FreeSWITCH, OpenACD is a powerful call delivery system. It has email capability built in. It has a complete web based agent UI as well as web based management. An protobuf based TCP connection for agents is also available.
